# Future Roadmap

## High Priority

- [ ] **SEO & Discovery**
  - Generate `sitemap.xml` for search engines.
  - Generate `rss.xml` / `atom.xml` for feed readers.
  - Add Open Graph (OG) meta tags for social sharing.

- [ ] **Navigation & UX**
  - Implement a client-side Search bar (Command+K).
  - Add a sticky Table of Contents (TOC) to post layouts.
  - Add "Reading Time" estimate to post headers.

## Enhancements

- [ ] **Performance**
  - Re-enable image optimization (requires custom loader for static export or using a service like Cloudinary).
  
- [ ] **Content**
  - Add support for "Series" (grouping related posts).
  - Add "Related Posts" section at the bottom of articles.

- [ ] **Analytics**
  - Integrate a privacy-friendly analytics tool (e.g., Plausible, Fathom) or a simple script.
